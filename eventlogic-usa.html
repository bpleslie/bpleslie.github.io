EventLogic USA is a relatively young marketing agency focused on experiential marketing. Specifically, setting up activations at events targeted for their clients.

I had the pleasure of creating the first iterations of eventlogicusa.com, which began with a very simple HTML5 / Bootstrap site from Initializr. The site has since evolved into a static HTML5 / Jekyll site hosted on GitHub via Github Pages.
